Perguntas Frequentes

O que é o markup BSP?
BSP (Business Solution Provider) é o parceiro certificado pela Meta que fornece acesso à API oficial do WhatsApp. O markup cobre os custos de infraestrutura, suporte, SLA e conformidade regulatória. No Brasil, os principais BSPs incluem Zenvia, Infobip, Twilio e outros parceiros locais.
Posso enviar marketing para qualquer número?
Não. Para mensagens de Marketing via API Oficial, o destinatário precisa ter dado opt-in explícito para receber comunicações da sua empresa. Enviar sem consentimento viola os Termos de Serviço da Meta e pode resultar em suspensão da conta WABA. Sempre colete opt-in de forma clara e documente.
Mensagens gratuitas dentro de 24h?
Sim! Quando um cliente envia uma mensagem para você, é aberta uma "janela de serviço" de 24 horas. Dentro dessa janela, você pode responder livremente com mensagens de Serviço sem custo de template. Após 24h sem resposta do cliente, você precisa usar um template aprovado (Utilidade ou Marketing).
Como integrar a API Meta?
Para usar a API Oficial (WABA - WhatsApp Business API), você precisa: (1) ter uma conta Meta Business verificada, (2) contratar um BSP parceiro, (3) registrar seu número de telefone dedicado, (4) criar e aprovar templates de mensagem no Meta Business Manager. O processo leva de 2 a 7 dias úteis.
Há desconto por volume?
Para categorias de Utilidade e Autenticação, a Meta oferece descontos progressivos para volumes acima de 1 milhão de mensagens por mês. Para Marketing, o preço por mensagem é fixo independente do volume. Os BSPs também podem oferecer preços negociados para grandes volumes — consulte nosso time comercial.
Nota: A integração com a API Oficial Meta (WABA) será disponibilizada em breve através de parceiros BSP certificados. Atualmente o sistema utiliza WhatsApp Web via Evolution API (método não-oficial, limitado a 1 número por instância).

Integração API Meta (WABA)
Em desenvolvimento 🚀
A integração nativa com a WhatsApp Business API (WABA) oficial da Meta está em nosso roadmap para o segundo semestre de 2026.

O que você vai poder fazer com a integração:

  • Enviar campanhas para listas ilimitadas via API certificada
  • Templates aprovados pela Meta com rich media
  • Relatórios de entrega, leitura e resposta em tempo real
  • Múltiplos números gerenciados no mesmo painel
  • Conformidade total com LGPD e termos Meta
Atualmente o sistema usa Evolution API (WhatsApp Web) — funcional para uso moderado. Para campanhas em escala, a API Oficial é altamente recomendada.
